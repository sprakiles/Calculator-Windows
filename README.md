# Windows Calculator (MAUI)

A modern, retro-styled calculator application built with .NET MAUI for Windows.

## Features

- **Retro Design**: A beautiful "Cream/Dark Retro" theme with rounded buttons and smooth aesthetics.
- **Theme Switching**: Seamlessly toggle between Light and Dark modes with a dedicated button.
- **Standard Operations**: Perform basic arithmetic operations (+, -, *, /) with ease.
- **Advanced Features**: Includes bracket support `( )` for complex expressions.
- **Responsive UI**: Designed to look great on Windows 10 and 11.

## Getting Started

### Prerequisites

- .NET 8.0 SDK or later
- Visual Studio 2022 (with .NET MAUI workload) + You can use vscode + maui workload for windows only

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sprakiles/Calculator-Windows.git
   ```
2. Open `Calculator.sln` in Visual Studio.
3. Build and run the project.

## Technologies Used

- **.NET MAUI**: Cross-platform framework for building native device apps.
- **C#**: Backend logic.
- **XAML**: UI definition.
